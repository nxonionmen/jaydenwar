# Jayden War

Jayden War는 웹 브라우저에서 바로 실행 가능한 2D RPG 어드벤처 게임입니다.
서버 설치 없이 `index.html` 파일만 열면 바로 플레이할 수 있습니다.

## 🎮 게임 시작하기
1. 이 저장소를 다운로드합니다.
2. 폴더 내의 `index.html` 파일을 더블 클릭하여 실행합니다.
3. 모험을 시작하세요!

## 🕹️ 조작법
- **이동 / 모드 전환**:
    - `H`: 집 (치료/휴식)
    - `B`: 전투 (적 선택)
    - `S`: 상점 (아이템 구매)
- **전투**:
    - `스페이스바`: 공격
    - `1` ~ `4`: 무기 교체
- **액션**:
    - `R`: 휴식 (집, 체력 낮을 때)
    - `P`: 포션 구매 (상점)
    - `G`: 대검 구매 (상점)
