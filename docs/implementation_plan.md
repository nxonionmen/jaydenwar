# 구현 계획 - 적 난이도 선택

사용자가 점점 강해지는 적을 선택해서 싸울 수 있는 기능을 추가합니다.

## 제안된 변경 사항

### 게임 로직
#### [수정] [src/game/Game.js](file:///c:/Users/onionmen/OneDrive/Documents/Projects/jayden_war/src/game/Game.js)
- **새로운 상태**: `BATTLE_SELECT` 추가. (전투 진입 전 단계)
- **적 데이터 확장**:
    - **오크**: HP 10, 공격력 1, 보상 20G (기존)
    - **기사**: HP 30, 공격력 2, 보상 50G
    - **드래곤**: HP 100, 공격력 5, 보상 200G

### UI 및 인터랙션
- **전투 진입 변경**: 'B' 키를 누르면 바로 전투가 시작되지 않고, **전투 선택 화면**으로 이동합니다.
- **선택 화면**:
    - (1) 오크 (쉬움)
    - (2) 기사 (보통)
    - (3) 드래곤 (어려움)
- **보상 시스템**: 적의 난이도에 따라 골드 보상을 차등 지급하도록 `winBattle` 로직 수정.

## 검증 계획
- 'B' 키 입력 시 선택 화면 진입 확인.
- 1, 2, 3 키로 각각 다른 적 생성 확인.
- 승리 시 난이도에 맞는 골드 획득 확인.
